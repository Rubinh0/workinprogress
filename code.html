What does 1984.py does?

1984.py is a simple TUI (text-based User Interface) programm that helps estimating your progress in reading the book 1984.

How did I made it?

This programm needed a few things:

  • setting the length of the book (in my case it's from page 11 to page 439)
  • taking the input (at which page you're at)
  • doing the pourcentage of read pages
  • based on the input estimaing how many time you will need til you finish the book
  • setting a progress bar
  • doing ASCII art to make it prettier
